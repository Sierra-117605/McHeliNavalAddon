## トラブルシューティング

| 症状 | 対処 |
|---|---|
| 偽装テクスチャが真っ黒 / 紫 | 偽装したブロックのリソースパック競合。別のブロックを試す |
| 左クリックでブロックが壊れる | Forgeイベントのキャンセルが効いていない。ログを確認 |
| GUIが開かない | F3+Tでリロード。それでもだめならログ確認 |
| エレベーターが動かない | フロアマーカーがコントローラーと同じX/Z座標にあるか確認 |
| クラッシュ | crash-reports/ のログを次回セッションで共有 |
