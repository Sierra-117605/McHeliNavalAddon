## プロジェクト概要

- **MOD名**: McHeliNavalAddon
- **対象**: Minecraft 1.14.4 / Forge
- **言語**: Java 8
- **依存MOD**: MCHELI（必須・リフレクション経由でアクセス）
- **MCHELIバージョン**: 1.14
- **設計方針**: 自律飛行なし。プレイヤーが手動操縦する前提の甲板装備をブロックとして追加する

## F1: カタパルト

### 概要
ブロック上の機体を前方へ強制加速して打ち出す。

### 動作仕様
- カタパルトブロックを複数並べてレール状に設置
- 機体がブロック上にいる状態で右クリック（またはレバー連動）でアクティベート
- ブロックのfacing方向へ機体の速度を強制設定
- 打ち出し速度は `config/mchelinaval.toml` で変更可能（デフォルト: TBD ※MCHELI解析後）
- 発射時にスチームパーティクル＋効果音
- 発射と連動してJBDに通知する

### 受け入れ条件
- [ ] カタパルトブロックを右クリックで機体が前方へ打ち出される
- [ ] 打ち出し方向がブロックの向きに従う
- [ ] 速度値をconfigで変更できる
- [ ] スチームエフェクトが出る

## F2: アレスティングワイヤー

### 概要
着艦時に機体を急減速・停止させる横断ワイヤー。

### 動作仕様
- ワイヤーブロックを横一列に3ブロック並べて設置（ワイヤー1本分）
- 機体が高速でブロック上を通過した瞬間に速度を段階的に0へ収束
- 減速はn tick（デフォルト20）かけて適用（自然な制動感）
- 速度閾値以下（地上走行速度相当）では作動しない
- キャッチ時にエフェクト＋効果音

### 受け入れ条件
- [ ] 着艦速度の機体がワイヤーを踏むと制動がかかり停止する
- [ ] 低速時は作動しない
- [ ] 制動エフェクト・効果音が出る

## F3: JBD（ジェットブラストデフレクター）

### 概要
カタパルト後方に設置する偏向板。機体のセット・射出状態に連動して自動で開閉する。

### 動作シーケンス
```
① 機体がカタパルトブロック上に乗る   → JBD 上がる（立ち上がり状態）
② カタパルト発射                     → そのまま上がったまま維持
③ 機体がカタパルトブロックから離れる  → JBD 閉じる（格納状態）
```

### 動作仕様
- カタパルトブロックに隣接して設置
- カタパルトから「機体セット」通知を受けてBlockStateを「立ち上がり」に変化
- カタパルトから「機体離脱」通知を受けてBlockStateを「格納」に変化
- 機体の離脱検知: カタパルトブロック上にMCHELI機体エンティティがいなくなった瞬間
- 立ち上がり中はコリジョンあり（通過不可）

### 受け入れ条件
- [ ] 機体がカタパルトに乗るとJBDが立ち上がる
- [ ] 機体が射出されカタパルトから離れるとJBDが格納される
- [ ] 立ち上がり中はブロックとして機能する

## F5: 移動プラットフォーム（UGOCRAFT式）

### 概要
コントローラーブロック1個で、周囲のブロック構造を丸ごと動かす汎用システム。
縦移動でエレベーター、横移動でスライドドアとして機能する。

### 動作仕様
- **移動コントローラーブロック**を構造物に設置
- コントローラーの設定項目:
  - 移動方向（X / Y / Z）
  - 移動距離（ブロック数）
  - 移動速度（毎tickの移動量、config変更可）
  - 対象範囲（コントローラーから何ブロック以内を動かすか）
- 右クリックで移動開始・停止トグル
- 移動対象: 設定範囲内の全ブロック＋その上にいるエンティティ（機体・プレイヤー含む）
- 移動中は周囲にエフェクト

### 使用例
| 用途 | 移動方向 | 備考 |
|---|---|---|
| デッキエレベーター | Y（縦）| 機体ごと甲板↔格納庫間を昇降 |
| スライドドア | X または Z（横）| 格納庫入口の開閉 |
| ランプ・ドック | 斜め | 応用次第で自由に |

### 受け入れ条件
- [ ] コントローラーを右クリックで構造物が動く
- [ ] 機体・プレイヤーも一緒に移動する
- [ ] 縦移動でエレベーターとして機能する
- [ ] 横移動でスライドドアとして機能する

## 非機能要件

| 項目 | 内容 |
|---|---|
| パフォーマンス | 移動プラットフォームの処理は10tick毎に間引く |
| 設定ファイル | `config/mchelinaval.toml`（Forge 1.14.4標準形式）|
| MCHELIアクセス | リフレクション経由。try-catchで保護。null安全徹底 |
| サーバー/クライアント | サーバーサイド処理主体。エフェクトはクライアント同期 |

## MCHELI解析が必要な事項（実装前に確認）

| # | 確認項目 | 用途 |
|---|---|---|
| 1 | 機体エンティティの基底クラス名 | フック対象の特定 |
| 2 | 速度フィールド名・型（XYZ成分か単一スカラーか）| カタパルト・ワイヤーの実装 |
| 3 | 機体がブロック上にいるかの判定方法 | カタパルト発火トリガー・JBD連動 |
| 4 | 速度書き換え後に挙動が変わるか | 実装可否の確認 |
